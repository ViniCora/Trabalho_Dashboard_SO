### Comandos
```
Informações detalhadas sobre todos os cpu
cat /proc/cpuinfo

Quantidade de cpu
nproc

Informações de hardware
lscpu

Informações sobre memória (free é resumido)
cat /proc/meminfo
free

Processos e porcentagens de uso de cpu e memoria
ps aux

Partições
cat /proc/partitions

Informações do sistema operacional
uname -a 
```
